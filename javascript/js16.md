### Javascript 面试题 - 性能优化

性能优化原则

- 多使用内存、缓存或者其他方法
- 减少 CPU 计算量，减少网络加载耗时
- （适用于所有编程的性能优化 - 空间换时间）

> 从何入手

- 让加载更快

1. 压缩资源体积： 压缩代码；
2. 减少代码访问次数：合并代码(合并 js 代码、css 代码、雪碧图)，ssr 服务端渲染，缓存(更多地命中缓存)；
3. 使用更快的网络：cdn;

- 让渲染更快

1. css 放在 head 内，js 文件放在 body 下；
2. 尽快地开始执行 js，用 DOMContentLoaded 触发；
3. 懒加载（图片懒加载，上滑动加载更多）；
